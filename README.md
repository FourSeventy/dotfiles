## Setup
Symlink dotfiles

cd ~

ln -s .dotfiles/.bash_profile ~/.bash_profile

ln -s  .dotfiles/.bashrc ~/.bashrc

ln -s  .dotfiles/.curlrc ~/.curlrc

ln -s  .dotfiles/.gitconfig ~/.gitconfig
